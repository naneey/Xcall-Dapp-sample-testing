# xCall Dapp Sample Testing

This repository contains sample dapp for icon and solidity.

## Deploying the contract

To compile and deploy contract for icon
- OpenJDK 11 
- gradle

To compile and deploy contract for solidity you can use [REMIX](https://remix.ethereum.org/)

## Interact with the deployed contract

- Icon: [goloop engine](https://github.com/icon-project/goloop/tree/master)
- Solidity : [REMIX](https://remix.ethereum.org/)
